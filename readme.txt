== Description ==

WordPress plugin for managing themes and plugins from github.

== Installation ==

= Installation =
1. Upload `git2wp.zip` to the `/wp-content/plugins/` directory;
2. Extract the `git2wp.zip` archive into the `/wp-content/plugins/` directory;
3. Activate the plugin through the 'Plugins' menu in WordPress.

Alternatively go into your WordPress dashboard and click on Plugins -> Add Plugin and search for Git2WP. Then click on Install, then on Activate Now.

= Usage =
Install and activate the plugin according with the Installation instructions.
