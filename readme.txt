== Description ==

WordPress plugin for managing themes and plugins from Github.

== Installation ==

= Installation =
1. Upload `git2wp.zip` to the `/wp-content/plugins/` directory;
2. Extract the `git2wp.zip` archive into the `/wp-content/plugins/` directory;
3. Activate the plugin through the 'Plugins' menu in WordPress.

Alternatively go into your WordPress dashboard and click on Plugins -> Add Plugin and search for GitHub2WP. Then click on Install, then on Activate Now.

= Usage =
Install and activate the plugin according with the Installation instructions.

== Frequently Asked Questions ==

= Why should I use this plugin? =
If you want to manage your plugins/themes keept on Github, this will help you to do this.

= Submodules not updating =
If you are using submodules you should know that the plugin only fetches level 1 submodules only.
